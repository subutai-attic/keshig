# Keshig
## About
### Installation
Obtain snappy-kvm.sh and storage.sh

As root user execute ```./snappy-kvm.sh``` in order to erect *Subutai RH* in KVM

Script will download following dependencies:

    - libvirt-bin
    - qemu-kvm
    - virtinst
    - Ubuntu Snappy Image(once)
    - Snappy-subutai repo
    - Snappy-tools
